# Cultify - Culture Discovery Platform

Cultify is a premium digital library for movies, series, books, and games. This project is divided into two main modules:

## 📁 Project Structure

- **/frontend**: Contains the user interface, styling, and client-side logic.
  - `index.html`: Main discovery page.
  - `pages/`: Authentication, Profile, and Settings pages.
  - `js/`: Application logic and components.
  - `css/`: Modern UI styles.
- **/backend**: (In development) Server-side logic and database integration.

## 🚀 Key Features

- Dynamic content discovery and filtering.
- User authentication and role-based access.
- Personalized watchlist management.
- Modern, responsive UI/UX design.
